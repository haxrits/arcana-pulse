# Digital Archive — Terrorist & Extremist Organizations

## Project info
Digital Archive — Terrorist & Extremist Organizations

An educational website that catalogs currently active extremist and terrorist organizations worldwide. Presents neutral, well-sourced, and factual information for academic and journalistic use, with cyber-themed UI and interactive organization profiles. No operational details or sensitive data included; all content is verified and cited from public sources.


## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
